### 创建钱包流程
1. 输入钱包名称
2. 选择助记词长度（12/24词）
3. 生成助记词
4. 确认已抄写
5. 完成

### 导入钱包流程
1. 输入钱包名称
2. 选择类型（助记词/私钥/WIF/XPRV）
3. 粘贴或输入私钥
4. 完成导入

## 🔐 安全说明

### 加密方式
- **算法**：AES-256-CBC
- **密钥派生**：PBKDF2（10,000次迭代）
- **存储位置**：Supabase（仅加密数据）
- **解密密钥**：仅存储在设备 Secure Enclave

### 安全等级
✅ 银行级加密  
✅ 生物识别保护  
✅ 防截屏保护  
✅ 自动清除剪贴板  
✅ 服务器无法解密  

## 📱 使用步骤

### 首次使用

1. **打开 App**
   - 点击底部 Tab 的比特币图标 ₿

2. **创建第一个钱包**
   - 点击 [创建钱包] 按钮
   - 输入钱包名称（如：主钱包）
   - 选择 12 或 24 词助记词
   - 点击 [生成助记词]
   - **重要**：抄写并妥善保管助记词
   - 勾选"我已抄写助记词"
   - 点击 [创建钱包]

3. **查看私钥**
   - 点击钱包卡片上的眼睛图标 👁️
   - 使用 Face ID / 指纹验证
   - 私钥将显示（此时截屏已被禁用）
   - 点击复制图标可复制（30秒后自动清除）

4. **备份钱包**
   - 点击云朵图标 ☁️
   - 确认备份
   - 系统将加密上传到云端

5. **导入现有钱包**
   - 点击 [导入钱包] 按钮
   - 输入钱包名称
   - 选择导入类型
   - 粘贴助记词或私钥
   - 点击 [导入钱包]
